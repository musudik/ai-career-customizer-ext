Privacy Policy - Last Updated: December 2024

Overview

AI Career Customizer is a browser extension that helps you create tailored resumes and cover letters using AI. We are committed to protecting your privacy and being transparent about how your data is handled.

Key Point: All your data is stored locally in your browser. We do not operate any servers and do not have access to your personal information.

Data We Collect

The extension stores the following data locally in your browser:

  • API Keys: Your AI provider API key (OpenAI, Anthropic, or Google)
  • Resume Data: Your uploaded or pasted resume content
  • Job Data: Temporarily captured job posting information
  • Profiles: Saved profile configurations
  • Settings: Your extension preferences

How Data is Transmitted

When you generate documents, the following data is sent to your selected AI provider:

  • The captured job posting content
  • Your resume content
  • Generation preferences (emphasis settings, tone)

Important: Data is sent directly from your browser to the AI provider's API using your own API key. We (the extension developers) never see or process this data.

Supported AI Providers

The extension supports multiple AI providers. Each has their own privacy policy governing how they handle API requests:

OpenAI Privacy Policy Anthropic Privacy Policy Google Privacy Policy

Data Storage & Security

  • Local Storage Only: All data is stored using Chrome's built-in storage API (chrome.storage.local), which encrypts data at rest
  • No External Servers: We do not operate any servers or databases
  • Secure Transmission: All API calls use HTTPS encryption
  • Consent Required: The extension asks for your consent before storing any data

PII Anonymization

The extension offers an optional PII (Personally Identifiable Information) anonymization feature that can:

  • Detect email addresses, phone numbers, and physical addresses in your resume
  • Replace this information with placeholders before sending to the AI provider
  • Help reduce exposure of sensitive personal data

This feature is optional and can be enabled in the Settings page.

Your Rights & Controls

You have full control over your data:

  • View Data: All stored data can be viewed in the Settings page
  • Delete Data: Use "Clear All Data" in Settings to remove everything
  • Revoke Consent: You can revoke consent at any time, which clears all stored data
  • Uninstall: Removing the extension deletes all associated data from your browser

Permissions

The extension requests only the minimum permissions necessary:

  • activeTab: To capture job posting content from the current page
  • storage: To save your settings, resume, and API key locally
  • scripting: To extract job information from web pages

Third-Party Services

This extension is a third-party tool and is not affiliated with OpenAI, Anthropic, or Google. When using AI features:

  • You are using your own API key and quota
  • The respective provider's terms of service and privacy policies apply
  • We recommend reviewing each provider's data handling practices

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected in the "Last Updated" date at the top of this page. Continued use of the extension after changes constitutes acceptance of the updated policy.

Contact

If you have any questions about this privacy policy or the extension's data practices, please open an issue on our GitHub repository or contact us through the Chrome Web Store listing.
